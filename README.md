# Edge AI - Projetos de Controle IoT com SLM

Este repositório contém dois projetos relacionados ao controle IoT utilizando **Small Language Models (SLM)** em aplicações de borda (Edge AI). Ambos os projetos integram inteligência artificial para controle inteligente de dispositivos embarcados, utilizando a placa Franzininho WiFi.

## Arquitetura

O repositório é dividido em dois projetos principais:

1. **SLM IoT Control**:
   - Sistema básico de controle IoT com SLM via API.
   - Backend simples em Flask.
   - Firmware embarcado para Franzininho WiFi com sensores e LEDs.

2. **SLM IoT Control Local**:
   - Sistema avançado com SLM executado localmente.
   - Backend completo com agente IA, RAG e controle IoT.
   - Firmware com suporte a OLED e módulo Pomodoro.
   - Containerização com Docker.
